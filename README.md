FEATURES

- Smooth animated modals (Apply, Post Job, Recruiter Login, Seeker Login)
- Reusable modal system using data attributes
- Responsive layout for mobile, tablet, and desktop
- Modern job card UI with save & apply interaction
- Clean Tailwind CSS design

OVERVIEW
A responsive job portal interface featuring animated modals, modern job cards, and a clean Tailwind UI. Built entirely using pure HTML, Tailwind CSS, and Vanilla JavaScript.

USAGE
Clone or download the repository and open `index.html` in your browser, or use VS Code Live Server.
